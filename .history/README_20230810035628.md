# 📖 StoX <a name="about-project"></a>

**StoX** is a web application for stock market which fetch data on available stocks in the market and provides analysis.

- **Select stock list for details**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
- Node.js
- Code Editor

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/Paulinelali/metric-capstone.git
```

### Install

Install this project with:

```sh
  cd React-Metrics-webapp
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```

### Run tests

To run tests, run the following command:

```sh
  npm test
```

### Deployment

You can deploy this project using:

- GitHub pages or any other deployment services of your choice.
